# Day1 - Intro to NodeJS and NPM
1. **Installing node latest version and npm**
  ```sh
    curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -
    sudo apt-get install -y nodejs
  ```

  ```sh
    npm install --save-dev <package-name>
  ```
